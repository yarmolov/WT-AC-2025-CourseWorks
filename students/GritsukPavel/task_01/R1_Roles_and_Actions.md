# Вариант 32 — Поиск работы «Оффер где?» — Роли и действия

Краткое описание

MVP: Job, Stage, Note, Reminder, Company, User. UI отображает канбан-доску с вакансиями и список напоминаний; API: /jobs, /stages, /notes, /reminders. Приёмка (MVP): корректное отображение и управление вакансиями на канбане.

Роли и основные действия

- Администратор
  - CRUD для пользователей
  - Просмотр статистики по всем пользователям
  - Управление системными настройками
  - Просмотр журнала событий

- Пользователь
  - CRUD для своих вакансий (Job)
  - CRUD для компаний (Company)
  - Управление этапами (Stage) для вакансий
  - Создание и редактирование заметок (Note)
  - Управление напоминаниями (Reminder)
  - Просмотр канбан-доски и перемещение вакансий
  - Экспорт своих данных

- Система
  - Отправка уведомлений о напоминаниях
  - Автоматическое обновление статусов

Примеры сценариев

- Пользователь добавляет вакансию в компании, создаёт этапы → перемещает вакансию по этапам на канбане → добавляет заметки и напоминания → получает уведомление о приближающемся дедлайне.
- Пользователь просматривает канбан за последний месяц, видит статистику откликов по компаниям, экспортирует данные.
