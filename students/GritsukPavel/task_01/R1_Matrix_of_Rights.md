# Вариант 32 — Матрица прав (операция × роль)

Операции: Просмотр вакансий, Создать/Редактировать/Удалить Job, Управление этапами, Создать/Редактировать заметки, Управление напоминаниями, Просмотр канбана, Управление пользователями, Просмотр статистики.

Роли: Администратор, Пользователь, Система

Таблица (кто может что)

- Просмотр вакансий: Администратор (все), Пользователь (свои)
- Создать Job: Пользователь
- Редактировать/Удалить Job: Пользователь (свои)
- Управление Company: Пользователь (свои)
- Управление Stage: Пользователь (для своих вакансий)
- Создать/Редактировать Note: Пользователь (для своих вакансий)
- Создать/Редактировать Reminder: Пользователь (для своих вакансий)
- Просмотр канбана: Пользователь (свои вакансии)
- Перемещение вакансий на канбане: Пользователь (свои)
- Управление пользователями: Администратор
- Просмотр общей статистики: Администратор
- Отправка уведомлений: Система

Примечания:

- По умолчанию пользователь видит только свои вакансии, компании и напоминания (scope ограничение).
- Администратор имеет доступ ко всем данным в режиме чтения для модерации и аналитики.
