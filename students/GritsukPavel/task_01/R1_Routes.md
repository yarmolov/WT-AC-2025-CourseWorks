# Вариант 32 — Поиск работы «Оффер где?» — Маршруты (user flows)

1. Идентификация (вход в систему)

   - Пользователь: логин → дашборд с вакансиями и напоминаниями
   - Администратор: логин → админ-панель

2. Подготовка данных (Пользователь)

   - Заполнить профиль
   - Добавить компании (Company)
   - Создать вакансии (Job)
   - Настроить этапы (Stage) для каждой вакансии

3. Работа с вакансиями (Пользователь)

   - Добавить/редактировать/удалить Job
   - Перемещать вакансии между этапами на канбане
   - Добавлять заметки (Note) к вакансии

4. Управление напоминаниями (Пользователь)

   - Создать Reminder для вакансии
   - Просмотр списка активных напоминаний
   - Отметить напоминание как выполненное

5. Просмотр канбана (Пользователь)

   - Просмотр вакансий в формате канбан-доски
   - Фильтрация по компаниям, этапам, дате
   - Перетаскивание карточек между этапами

6. Обработка статусов

   - Пользователь обновляет статус вакансии
   - Система автоматически отслеживает историю изменений
   - Уведомления о приближающихся напоминаниях

7. Отчёт / Аналитика

   - Просмотр статистики откликов по компаниям
   - Экспорт данных для анализа
   - История взаимодействия с вакансиями
