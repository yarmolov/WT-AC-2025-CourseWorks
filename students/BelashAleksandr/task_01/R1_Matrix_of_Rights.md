# Вариант 37 — Матрица прав (операция × роль)

Операции: Просмотр ленты, Создать/Редактировать/Удалить Source, Создать/Редактировать/Удалить Tag, Работа с избранным, Подать жалобу, Модерация жалоб, Управление пользователями.

Роли: Администратор, Модератор, Пользователь

Таблица (кто может что)

- Просмотр ленты (Feed): Администратор, Модератор, Пользователь
- Просмотр статьи (Article): Администратор, Модератор, Пользователь
- Создать Source: Администратор
- Редактировать/Удалить Source: Администратор
- Создать Tag: Администратор
- Редактировать/Удалить Tag: Администратор
- Добавить в избранное (Favorite): Пользователь
- Удалить из избранного: Пользователь
- Просмотр своего избранного: Пользователь
- Подать жалобу (Report): Пользователь
- Просмотр жалоб: Администратор, Модератор
- Рассмотрение/Закрытие жалобы: Администратор, Модератор
- Удаление статьи: Администратор, Модератор
- Управление пользователями: Администратор

Примечания:

- Пользователь видит только своё избранное (scope ограничение).
- Модератор не может управлять пользователями и источниками.
