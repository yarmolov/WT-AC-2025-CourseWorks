# Вариант 37 — Новости «Без фейков» — Маршруты (user flows)

1. Идентификация (вход в систему)

   - Пользователь: логин → лента новостей и избранное
   - Модератор: логин → панель модерации жалоб
   - Администратор: логин → админ-панель

2. Подготовка данных (Администратор)

   - Заполнить список пользователей
   - Создать источники новостей (Source)
   - Создать теги (Tag)

3. Работа с источниками (Администратор)

   - Добавить/редактировать/удалить Source
   - Добавить/редактировать/удалить Tag

4. Просмотр ленты (Пользователь)

   - Просмотр списка статей в ленте
   - Фильтрация по тегам и источникам
   - Просмотр детальной страницы статьи

5. Работа с избранным (Пользователь)

   - Добавление статьи в избранное
   - Просмотр списка избранного
   - Удаление статьи из избранного

6. Подача жалобы (Пользователь)

   - Открыть форму жалобы на статью
   - Указать причину жалобы
   - Отправить жалобу

7. Модерация (Модератор/Администратор)

   - Просмотр списка жалоб
   - Рассмотрение жалобы
   - Принятие решения (отклонить/удалить статью)
