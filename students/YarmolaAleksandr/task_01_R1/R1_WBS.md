# Вариант 09 — WBS (эпики → фичи → задачи)

- Эпик A. Модель данных и миграции
  - Фичи: схемы User/Post/Tag/Comment/Like; миграции Prisma
  - Задачи: A1. Описать таблицы и связи; A2. Написать Prisma схему; A3. Подготовить демо-данные

- Эпик B. API и аутентификация
  - Фичи: REST API (/auth, /posts, /comments, /likes); JWT авторизация
  - Задачи: B1. Реализовать endpoints CRUD для постов; B2. Реализовать authMiddleware; B3. Добавить пагинацию и фильтры

- Эпик C. Frontend структура и роутинг
  - Фичи: React Router, страницы (лента, пост, редактор, черновики); компоненты PostView
  - Задачи: C1. Настроить React Router; C2. Создать PostPage и PostList; C3. Реализовать форму редактора

- Эпик D. Функционал постов и взаимодействий
  - Фичи: CRUD постов, комментарии, лайки, теги; черновики
  - Задачи: D1. Реализовать likePost/addComment; D2. Поддержка черновиков (published=false); D3. UI для тегов

- Эпик E. Безопасность и роли
  - Фичи: RBAC (роли admin/user), scope доступ, middleware защита
  - Задачи: E1. Реализовать проверки прав в роутерах; E2. Тесты доступа (owner/admin); E3. Документация прав

- Эпик F. Окружение и деплой
  - Фичи: Docker, CI/CD pipeline, Swagger документация, тесты
  - Задачи: F1. Dockerfile для backend/frontend; F2. Github Actions/CI; F3. Подготовить инструкции для демо
