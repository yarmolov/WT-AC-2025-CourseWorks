# Вариант 1 — Task Manager — Маршруты (user flows)

1. Идентификация (вход в систему)
   - Пользователь: логин → доска задач (Kanban)
   - Администратор: логин → админ-панель с отчётами

2. Создание и управление задачами (Менеджер/Админ)
   - Создать новую задачу
   - Назначить исполнителя
   - Установить приоритет и сроки

3. Работа с задачами (Пользователь)
   - Просмотр назначенных задач
   - Изменение статуса (New → In Progress → Review → Completed)
   - Добавление комментариев
   - Загрузка файлов

4. Отслеживание прогресса
   - Просмотр доски Kanban
   - Фильтрация задач по статусу, приоритету, исполнителю
   - Поиск по названию задачи

5. Уведомления
   - Получение уведомлений о новых задачах
   - Уведомления об изменениях статуса
   - Уведомления о новых комментариях

6. Отчётность
   - Генерация отчётов по выполненным задачам
   - Аналитика по производительности команды
   - Экспорт данных
