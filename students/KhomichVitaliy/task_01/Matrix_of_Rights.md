# Вариант 1 — Матрица прав (операция × роль)

Операции: Просмотр задач, Создать/Редактировать/Удалить Task, Просмотр пользователей, Назначение задач, Комментирование, Загрузка вложений, Просмотр отчётов.

Роли: Администратор, Менеджер, Пользователь

Таблица (кто может что)

- Просмотр всех задач: Администратор, Менеджер
- Просмотр своих задач: Администратор, Менеджер, Пользователь
- Создать Task: Администратор, Менеджер
- Редактировать/Удалить Task: Администратор, Менеджер (свои задачи)
- Назначение задач: Администратор, Менеджер
- Изменение статуса задачи: Администратор, Менеджер, Пользователь (свои задачи)
- Добавление комментариев: Администратор, Менеджер, Пользователь (участники задачи)
- Загрузка вложений: Администратор, Менеджер, Пользователь (участники задачи)
- Просмотр пользователей: Администратор
- Создание/Редактирование пользователей: Администратор
- Просмотр отчётов: Администратор, Менеджер
