# Вариант 35 — Поездки «Поехали!» — Маршруты (user flows)

1. Идентификация (вход в систему)

   - Пользователь: логин → дашборд со списком поездок
   - Администратор: логин → админ-панель

2. Подготовка данных (Администратор)

   - Заполнить список пользователей
   - Настроить базовые параметры системы

3. Работа с поездками (Пользователь)

   - Создать новую поездку (Trip)
   - Добавить остановки (Stop) к поездке
   - Указать даты и бюджет
   - Добавить заметки (Note)
   - Редактировать/удалить поездку

4. Совместная работа (Пользователь)

   - Поделиться поездкой с другими пользователями
   - Пригласить участников в поездку
   - Просмотр совместных поездок

5. Просмотр и планирование (Пользователь)

   - Просмотр списка своих поездок
   - Просмотр деталей поездки: маршрут, остановки, даты
   - Просмотр бюджета и расходов (Expense)
   - Просмотр карты маршрута

6. Управление бюджетом (Пользователь)

   - Добавить расход к поездке
   - Отслеживание общего бюджета
   - Просмотр истории трат

7. Отчёт / История

   - Просмотр архива завершённых поездок
   - Экспорт данных о поездке
