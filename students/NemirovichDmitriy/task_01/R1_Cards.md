# Вариант 35 — Карточки (обязательные поля и валидации)

1. Форма аутентификации

- username: string, не пустое → ошибка: «Введите имя пользователя.»
- email: string, валидный email → ошибка: «Введите корректный email.»
- password: string, не пустое → ошибка: «Введите пароль.»

1. Карточка поездки (Trip)

- id: UUID, автогенерируется
- title: string, не пустое → ошибка: «Введите название поездки.»
- description: string, опционально
- start_date: date, не пустое → ошибка: «Укажите дату начала.»
- end_date: date, не пустое, должна быть >= start_date → ошибка: «Дата окончания должна быть после даты начала.»
- budget: number, опционально, >= 0 → ошибка: «Бюджет не может быть отрицательным.»
- owner_id: reference -> User.id, не пустое → ошибка: «Владелец поездки не указан.»

1. Карточка остановки (Stop)

- id: UUID, автогенерируется
- trip_id: reference -> Trip.id, не пустое → ошибка: «Выберите поездку.»
- name: string, не пустое → ошибка: «Введите название остановки.»
- address: string, опционально
- latitude: number, опционально, в диапазоне [-90, 90] → ошибка: «Неверная широта.»
- longitude: number, опционально, в диапазоне [-180, 180] → ошибка: «Неверная долгота.»
- arrival_date: datetime, опционально
- departure_date: datetime, опционально
- order: integer, не пустое, >= 0 → ошибка: «Порядок остановки должен быть >= 0.»

1. Карточка заметки (Note)

- id: UUID, автогенерируется
- trip_id: reference -> Trip.id, не пустое → ошибка: «Выберите поездку.»
- author_id: reference -> User.id, не пустое → ошибка: «Автор заметки не указан.»
- content: text, не пустое → ошибка: «Содержимое заметки не может быть пустым.»
- created_at: datetime, автогенерируется

1. Карточка расхода (Expense)

- id: UUID, автогенерируется
- trip_id: reference -> Trip.id, не пустое → ошибка: «Выберите поездку.»
- author_id: reference -> User.id, не пустое → ошибка: «Автор расхода не указан.»
- amount: number, не пустое, > 0 → ошибка: «Сумма расхода должна быть больше 0.»
- category: string, опционально
- description: string, опционально
- date: date, не пустое → ошибка: «Укажите дату расхода.»

1. Карточка участника поездки (TripParticipant)

- id: UUID, автогенерируется
- trip_id: reference -> Trip.id, не пустое → ошибка: «Выберите поездку.»
- user_id: reference -> User.id, не пустое → ошибка: «Выберите пользователя.»
- joined_at: datetime, автогенерируется

**Примечание:** Владелец поездки определяется через Trip.owner_id. Запись в TripParticipant означает только факт участия в поездке.
