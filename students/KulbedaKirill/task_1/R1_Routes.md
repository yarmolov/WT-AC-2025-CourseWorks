# Вариант 34 — HelpDesk — Маршруты (user flows)

1. Идентификация (вход в систему)

   - Пользователь: логин → карточки с тикетами
   - Администратор: логин → выбор коллекции (агенты, пользователь, очереди)

2. Подготовка данных (Администратор)

   - Открыть список пользователей
   - Создать пользователя

3. Работа с очередями (Администратор)

   - Добавить/редактировать/удалить Queue

4. Просмотр пользователем (Пользователь)

   - Просмотр списка открытых тикетов
   - Просмотр истории сообщений тикета
