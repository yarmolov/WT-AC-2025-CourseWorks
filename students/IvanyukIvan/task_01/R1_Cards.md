# Вариант 14 — Карточки (обязательные поля и валидации)

1. Форма аутентификации

- username: string, латиница + цифры + underscore, 3-30 символов, не пустое → ошибка: «Введите корректное имя пользователя (латиница, цифры, underscore, 3-30 символов).»
- email: string, валидный email, не пустое → ошибка: «Введите корректный email.»
- password: string, минимум 8 символов, не пустое → ошибка: «Пароль должен содержать минимум 8 символов.»

1. Карточка темы (Topic)

- id: UUID, автогенерируется
- name: string, не пустое → ошибка: «Введите название темы.»
- description: string, опционально
- created_at: datetime, автогенерируется

1. Карточка цели (Goal)

- id: UUID, автогенерируется
- topic_id: reference -> Topic.id, не пустое → ошибка: «Выберите тему.»
- user_id: reference -> User.id, не пустое → ошибка: «Выберите пользователя.»
- name: string, не пустое → ошибка: «Введите название цели.»
- description: string, опционально
- target_value: number, не пустое → ошибка: «Введите целевое значение.»
- deadline: datetime, опционально

1. Карточка записи прогресса (ProgressEntry)

- id: UUID, автогенерируется
- goal_id: reference -> Goal.id, не пустое → ошибка: «Выберите цель.»
- timestamp: datetime, не пустое → ошибка: «Выберите дату и время.»
- value: number, не пустое → ошибка: «Неверный формат значения.»
- comment: string, опционально, max 5000

1. Карточка напоминания (Reminder, НЕ входит в MVP, только бонус)

- id: UUID, автогенерируется
- goal_id: reference -> Goal.id, не пустое → ошибка: «Выберите цель.»
- reminder_time: datetime, не пустое → ошибка: «Выберите время напоминания.»
- message: string, не пустое → ошибка: «Сообщение напоминания не может быть пустым.»

**Примечание:** Сущность Reminder реализуется только при наличии времени после MVP. В минимальной версии приложения она отсутствует.
