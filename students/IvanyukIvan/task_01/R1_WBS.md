# Вариант 14 — WBS (эпики → фичи → задачи)

- Эпик A. Модель данных и миграции
  - Фичи: схемы Topic/Goal/ProgressEntry/User; миграции
  - Задачи: A1. Описать таблицы и связи; A2. Написать миграции; A3. Подготовить демо‑данные

- Эпик B. API и интеграция данных
  - Фичи: REST API (/topics, /goals, /progress, /reports)
  - Задачи: B1. Реализовать endpoints CRUD; B2. Реализовать добавление прогресса; B3. Добавить пагинацию и фильтры

- Эпик C. Расчёт прогресса и отчёты
  - Фичи: вычисление процента выполнения, генерация отчётов, статистика
  - Задачи: C1. Реализовать подсчёт прогресса; C2. Тесты на корректность расчётов; C3. UI для просмотра отчётов

- Эпик D. UI / диаграммы и графики
  - Фичи: дашборд, диаграммы прогресса, список целей
  - Задачи: D1. Компонент диаграммы прогресса; D2. Страница целей; D3. Личный дашборд с графиками

- Эпик E. Безопасность и роли
  - Фичи: аутентификация, авторизация, scope доступа
  - Задачи: E1. Реализовать RBAC; E2. Тесты доступа; E3. Документация прав

- Эпик F. Окружение и развёртывание (MVP: без Docker/K8s/CI)
  - Фичи MVP: локальный запуск, README с инструкциями, .env.example
  - Задачи MVP: F1. Настроить локальную БД PostgreSQL; F2. Создать .env.example; F3. Написать README с инструкциями запуска
