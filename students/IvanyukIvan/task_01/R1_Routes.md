# Вариант 14 — Прогресс «До сессии успею» — Маршруты (user flows)

1. Идентификация (вход в систему)

   - Пользователь: логин → дашборд с темами и прогрессом
   - Администратор: логин → админ-панель

2. Подготовка данных (Администратор)

   - Заполнить список пользователей
   - Создать темы (Topic)
   - Создать цели (Goal)
   - Настроить связи Topic ↔ Goal

3. Работа с темами и целями (Администратор)

   - Добавить/редактировать/удалить Topic или Goal
   - Назначить цели пользователям

4. Работа с прогрессом (Пользователь)

   - Просмотр списка тем и целей
   - Добавление записей о прогрессе (ProgressEntry)
   - Отметка выполнения задач

5. Просмотр и анализ (Пользователь)

   - Просмотр личного дашборда с прогрессом
   - Просмотр диаграмм и графиков прогресса
   - Просмотр списка целей с процентом выполнения

6. Формирование отчётов

   - Пользователь/Администратор генерирует отчёт
   - Просмотр статистики по темам и целям
   - Экспорт данных для анализа
