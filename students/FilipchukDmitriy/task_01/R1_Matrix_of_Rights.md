# Вариант 16 — Матрица прав (операция × роль)

Операции: Просмотр тетрадей, Создать/Редактировать/Удалить Notebook, Создать/Редактировать/Удалить Note, Управление метками, Управление доступом (Share), Управление пользователями.

Роли: Администратор, Пользователь

Таблица (кто может что)

- Просмотр своих тетрадей: Администратор, Пользователь
- Просмотр чужих тетрадей (при наличии Share): Администратор, Пользователь
- Создать Notebook: Администратор, Пользователь
- Редактировать/Удалить свой Notebook: Администратор, Пользователь
- Редактировать/Удалить чужой Notebook: Администратор
- Создать Note: Администратор, Пользователь (в своих или расшаренных тетрадях)
- Редактировать/Удалить Note: Администратор, Пользователь (владелец или при наличии доступа)
- Создать/Редактировать/Удалить Label: Администратор, Пользователь
- Создать/Редактировать/Удалить Share: Администратор, Пользователь (владелец тетради)
- Управление пользователями: Администратор

Примечания:

- Пользователь видит только свои тетради и те, к которым есть доступ (scope ограничение).
- Уровень доступа в Share определяет, может ли пользователь редактировать или только просматривать.
