# Вариант 16 — Заметки «Пишем вдвоём» — Маршруты (user flows)

1. Идентификация (вход в систему)

   - Пользователь: логин → список тетрадей
   - Администратор: логин → админ-панель
   - Выход: logout (инвалидация токена)

2. Подготовка данных (Администратор)

   - Заполнить список пользователей
   - Создать системные метки (Label с is_system=true)

3. Работа с тетрадями (Пользователь)

   - Создать/редактировать/удалить Notebook
   - Добавить/редактировать/удалить Note в тетради (требуется permission=write для расшаренных)
   - Присвоить метки (Label) заметкам через /notes/{id}/labels

4. Совместный доступ (Пользователь)

   - Поделиться тетрадью с другим пользователем (Share)
   - Указать уровень доступа (read — только чтение / write — редактирование)
   - Отозвать доступ (владелец тетради)
   - Отказаться от доступа (получатель Share может удалить свой доступ)

5. Просмотр и редактирование (Пользователь)

   - Просмотр списка своих тетрадей
   - Просмотр тетрадей, к которым есть доступ
   - Редактирование заметок
   - Просмотр истории изменений

6. Управление метками (Пользователь)

   - Создать/редактировать/удалить свои метки
   - Фильтровать заметки по меткам

7. История изменений

   - Просмотр истории редактирования заметки
   - Восстановление предыдущей версии
