# Вариант 18 — Рецепты «Что приготовить?» — Маршруты (user flows)

1. Идентификация (вход в систему)

   - Гость: поиск рецептов по ингредиентам → просмотр деталей → регистрация для сохранения
   - Пользователь: логин → дашборд с планом питания на сегодня
   - Администратор: логин → панель управления справочниками (ингредиенты, теги)

2. Поиск и просмотр рецептов (Гость / Пользователь)

   - Просмотр ленты рецептов (новые/популярные)
   - Поиск по названию
   - Фильтр «Что в холодильнике»: выбор имеющихся ингредиентов → список подходящих рецептов
   - Фильтр по тегам (веганское, без глютена, острое)
   - Просмотр карточки рецепта (ингредиенты, шаги, КБЖУ)

3. Управление рецептами (Пользователь)

   - Создание своего рецепта (название, фото, шаги)
   - Выбор ингредиентов из справочника с указанием количества
   - Добавление тегов (диета, тип блюда)
   - Публикация рецепта (сразу доступен всем или премодерация — зависит от настроек)

4. Планирование питания (Пользователь)

   - Открытие календаря (Meal Plan)
   - Выбор даты и типа приема пищи (Завтрак, Обед, Ужин)
   - Добавление рецепта в слот (из избранного или поиска)
   - Удаление/перенос блюд в плане

5. Список покупок (Пользователь)

   - Просмотр текущего списка покупок
   - Автогенерация списка на основе Плана питания (выбранный период)
   - Ручное добавление товаров (не из рецептов, например «Молоко»)
   - Отметка купленных товаров (вычеркивание)

6. Управление справочниками (Администратор)

   - Добавление базовых ингредиентов (нормализация названий)
   - Управление тегами и категориями
   - Модерация жалоб на рецепты (спам, некорректный контент)

7. Профиль и настройки (Пользователь)

   - Настройка предпочтений (аллергии, исключаемые продукты)
   - Смена пароля/email
   - Просмотр своих опубликованных рецептов
