# Вариант 36 — Идеи «Лайк за мысль» — Роли и действия

Краткое описание
MVP: Idea, Tag, Comment, Vote, User. UI отображает рейтинг идей и систему комментариев; API: /ideas, /tags, /comments, /votes. Приёмка (MVP): корректный расчет и отображение рейтинга идей на основе голосов.

Роли и основные действия

- Администратор
  - CRUD для пользователей
  - CRUD для тегов (Tag) и модерация идей (Idea)
  - Одобрение/отклонение новых идей
  - Управление комментариями (удаление, редактирование)
  - Контроль за соблюдением правил сообщества

- Пользователь
  - Просмотр списка идей с сортировкой по рейтингу/дате
  - Создание и редактирование своих идей
  - Голосование (Vote) за идеи других пользователей
  - Добавление комментариев (Comment) к идеям

Примеры сценариев:
- Пользователь создает Idea с тегами → система публикует идею после модерации → другие пользователи голосуют и комментируют → рейтинг идеи автоматически пересчитывается.
- Пользователь просматривает топ идей по рейтингу за неделю, оставляет комментарий и голосует за понравившуюся идею; система обновляет рейтинг в реальном времени.
