# Вариант 20 — Микро-твиттер «Мысли вслух» — Маршруты (user flows)

## 1. Идентификация (вход в систему)

- Пользователь: логин → попадает в ленту (Feed), где отображаются посты пользователей, на которых он подписан.
- Администратор: логин → переходит в админ-панель (список пользователей, жалоб, модерация контента).

## 2. Подготовка данных (Администратор)

- Создать пользователей (опционально — через админку).
- Просмотреть список всех постов.
- Просмотреть список жалоб (Reports).
- Настроить модерацию: отмечать жалобы, удалять нарушающие посты/ответы.

## 3. Работа пользователя (Пользователь)

- Создать новый пост (до 280 символов).
- Редактировать собственный пост.
- Удалить собственный пост.
- Подписаться на другого пользователя.
- Отписаться от пользователя.
- Поставить лайк посту / убрать лайк.
- Ответить на пост (Reply).
- Посмотреть ответы на пост.
- Отправить жалобу (Report) на пост в случае нарушения правил.

## 4. Работа с лентой (Пользователь)

- Открыть главную страницу → загрузить ленту постов от подписок.
- Пролистывать/подгружать посты (пагинация/бесконечный скролл).
- По клику на автора → переход в профиль пользователя:
  - Посты автора
  - возможность подписаться/отписаться
  - счётчики подписчиков/подписок

## 5. Работа с жалобами (Администратор)

- Просмотр списка всех жалоб (Report):
  - фильтр по статусу (open/reviewed).
- Просмотр деталей жалобы:
   -пост, на который подана жалоба
   -причина
   -автор жалобы
- Действия:
   -удалить пост
   -пометить жалобу как reviewed

## 6. Профиль пользователя

- Пользователь видит свой профиль:
  - свои посты
  - свои подписки
  - своих подписчиков
  - настройки аккаунта (username, email)

- Другие пользователи видят:
  - чужие посты
  - подписки/подписчиков
  - кнопку «Подписаться» / «Отписаться»

## 7. Уведомления (опционально)

- Уведомление при новом подписчике
- Уведомление о новом ответе на ваш пост
- Уведомление о лайке
- Уведомление администратору о новой жалобе

## 8. История и модерация (Администратор)

- Просмотр удалённых постов
- Просмотр истории действий модераторов
- Просмотр активности нарушителей

## Итоговая карта маршрутов

| Роль          | Основные маршруты                                                         |
| ------------- | -------------------------------------------------------------------------- |
| Гость         | `/login`, `/register`                                                     |
| Пользователь  | `/feed`, `/post/:id`, `/profile/:username`, `/settings`, `/likes`, `/replies` |
| Администратор | `/admin`, `/admin/users`, `/admin/reports`, `/admin/posts`, `/admin/logs` *(опционально)* |
