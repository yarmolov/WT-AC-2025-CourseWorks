# Вариант 20 — Матрица прав (операция × роль)

Операции:
- Просмотр постов
- Создать пост
- Удалить пост
- Просмотр ленты (Feed)
- Подписка / Отписка (Follow/Unfollow)
- Лайк / Удалить лайк
- Создать ответ (Reply)
- Просмотр ответов
- Создать репорт (Report)
- Просмотр/Обработка репортов
- Управление пользователями
- Просмотр журналов модерации (опционально)
- Роли: Администратор, Пользователь, Сервис (system)

Таблица (кто может что):
- Просмотр постов: Администратор, Пользователь
- Создать пост: Пользователь
- Редактировать пост: Пользователь (только свои)
- Удалить пост: Пользователь (только свои), Администратор (любые)
- Просмотр ленты (Feed): Пользователь
- Подписаться / Отписаться: Пользователь
- Лайк / Удалить лайк: Пользователь
- Создать ответ (Reply): Пользователь
- Просмотр ответов: Администратор, Пользователь
- Просмотр/Обработка репортов: Администратор
- Управление пользователями: Администратор
- Просмотр журналов/логов: Администратор

Примечания:

- По умолчанию пользователь видит только свои устройства/дома (scope ограничение).
- Некоторые операции (например, закрытие Alert) могут быть ограничены политикой: только Администратор или владелец устройства.
