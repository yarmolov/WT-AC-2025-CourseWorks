# Вариант 27 — Pomodoro «Томатная продуктивность» — Маршруты (user flows)

1. Идентификация (вход в систему)

   - Пользователь: логин → дашборд с задачами и статистикой
   - Администратор: логин → админ-панель

2. Подготовка данных (Пользователь)

   - Создать задачи (Task)
   - Создать теги (Tag)
   - Назначить теги задачам

3. Работа с задачами (Пользователь)

   - Добавить/редактировать/удалить Task
   - Назначить приоритет задаче
   - Отфильтровать задачи по тегам

4. Работа с Pomodoro-сессиями (Пользователь)

   - Запустить таймер для выбранной задачи
   - Пауза/возобновление сессии (PATCH /sessions/{id}/pause, /resume)
   - Завершение сессии (PUT /sessions/{id})
   - Отметить сессию как завершенную или прерванную
   - Получить уведомление при завершении сессии

5. Настройки пользователя

   - Просмотр/изменение настроек уведомлений (push/email/sound)

6. Просмотр статистики (Пользователь)

   - Просмотр количества завершенных сессий за день/неделю/месяц
   - Просмотр времени работы по задачам
   - Просмотр времени работы по тегам
   - Просмотр графиков продуктивности

7. Управление пользователями (Администратор)

   - Просмотр списка пользователей
   - Создание/редактирование/удаление пользователей
   - Просмотр статистики по всем пользователям
   - Экспорт данных любого пользователя (с параметром userId)

8. Отчёты

   - Экспорт статистики за период
   - Просмотр истории сессий
