# Вариант 27 — Матрица прав (операция × роль)

Операции: Просмотр задач, Создать/Редактировать/Удалить Task, Запуск/Пауза Session, Просмотр Session, Создать/Редактировать/Удалить Tag, Просмотр статистики, Управление пользователями, Экспорт данных.

Роли: Администратор, Пользователь

Таблица (кто может что)

- Просмотр задач: Администратор (все), Пользователь (свои)
- Создать Task: Пользователь
- Редактировать Task: Пользователь (свои)
- Удалить Task: Пользователь (свои)
- Запуск Session: Пользователь
- Пауза/Возобновление Session: Пользователь (свои)
- Завершение Session: Пользователь (свои)
- Удалить Session: Пользователь (свои), Администратор (все)
- Просмотр Session: Администратор (все), Пользователь (свои)
- Создать/Редактировать/Удалить Tag: Пользователь (свои)
- Просмотр статистики: Администратор (все пользователи), Пользователь (своя)
- Управление пользователями: Администратор
- Экспорт данных: Пользователь (своих), Администратор (всех, с параметром userId)
- Настройки уведомлений: Пользователь (свои)

Примечания:

- По умолчанию пользователь видит только свои задачи, сессии и теги.
- Администратор имеет полный доступ ко всем данным для аналитики и управления.
