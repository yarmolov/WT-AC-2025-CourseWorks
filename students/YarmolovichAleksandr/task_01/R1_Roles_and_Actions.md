# Вариант 38 — Объявки: бери, пока горячее» — Роли и действия

Краткое описание

MVP: User, Category, Ad, Media, Conversation, Message, Report.
UI отображает список объявлений, фильтрацию по категориям, карточку объявления и чат с продавцом.
API: /ads, /categories, /users, /conversations, /messages, /reports, /uploads.
Приёмка (MVP): корректное создание объявления, возможность отправить сообщение продавцу, корректная подача и обработка жалобы.

Роли и основные действия

- Администратор
  - CRUD для пользователей,
  - CRUD для категорий,
  - Управление объявлениями: редактирование, удаление, блокировка,
  - Просмотр и обработка жалоб,
  - Просмотр журнала действий и статистики (опционально).

- Модератор
  - Просмотр всех объявлений,
  - Просмотр и обработка жалоб,
  - Блокировка и разблокировка объявлений,
  - Редактирование или удаление объявлений при нарушениях.

- Пользователь
  - Просмотр списка объявлений и категорий,
  - Создание объявления и загрузка медиа,
  - Редактирование и удаление своих объявлений,
  - Просмотр профилей других пользователей,
  - Создание чата с продавцом и отправка сообщений,
  - Подача жалобы на объявление.

- Сервис / Интеграция
  - Обработка загрузок медиа, включая сохранение изображений и проверку их корректности.
  - Автоматическая модерация объявлений (проверка текста и изображений на запрещённый контент, стоп-слова, потенциальный спам).
  - Автоматическая генерация жалоб/флагов при нарушении правил (например, если объявление содержит запрещённые слова или подозрительную активность).

Примеры сценариев

- Пользователь создаёт объявление → загружает фотографии → объявление появляется в списке в своей категории.
- Покупатель открывает объявление → нажимает “Написать” → создаётся чат → участники обмениваются сообщениями.
- Пользователь заполняет форму жалобы → модератор получает новое обращение → проверяет контент → блокирует объявление или отклоняет жалобу.
- Админ добавляет новую категорию → проверяет список пользователей → выдаёт модератору новую роль → просматривает статистику по жалобам и активности.
