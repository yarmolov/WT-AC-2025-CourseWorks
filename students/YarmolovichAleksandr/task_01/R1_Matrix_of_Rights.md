# Вариант 38 — Матрица прав (операция × роль)

Операции: просмотр объявлений, создание/редактирование/удаление объявления, управление категориями, загрузка медиа, просмотр и отправка сообщений, обработка жалоб, управление пользователями, просмотр служебной информации.

Роли: Администратор, Модератор, Пользователь

Таблица (кто может что)

- Просмотр объявлений: Администратор, Модератор, Пользователь
- Создать объявление: Пользователь, Модератор, Администратор
- Редактировать объявление: Пользователь (только свои), Модератор, Администратор
- Удалить объявление: Пользователь (только свои), Модератор, Администратор
- Блокировка объявления: Модератор, Администратор
- Управление категориями: Администратор
- Загрузка медиа: Пользователь, Модератор, Администратор
- Создание чата: Пользователь, Модератор, Администратор
- Отправка сообщений: Пользователь, Модератор, Администратор
- Просмотр жалоб: Модератор, Администратор
- Создать жалобу: Пользователь, Модератор, Администратор
- Обработка жалоб (смена статуса): Модератор, Администратор
- Управление пользователями: Администратор
- Просмотр журналов/служебных данных: Администратор

Примечания:

- По умолчанию пользователь видит только свои устройства/дома (scope ограничение).
- Некоторые операции (например, закрытие Alert) могут быть ограничены политикой: только Администратор или владелец устройства.
