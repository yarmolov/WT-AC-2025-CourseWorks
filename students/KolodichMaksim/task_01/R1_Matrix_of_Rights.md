# R1 — Матрица прав (операция × роль)

Операции / Роли: Пользователь (User), Сервис (system), Администратор (optional)

- Просмотр метрик: User, Admin
- Создать метрику: User
- Редактировать/Удалить метрику: User (только свои)
- Просмотр записей (Entries): User, Admin
- Создать запись (Entry): User, System
- Редактирование/Удаление записи: User (только свои)
- Просмотр дашборда/отчётов: User, Admin
- Управление пользователями: Admin

Примечания:
- Все операции требуют аутентификации (JWT), кроме `register`/`login`.
- Scope: пользователь видит только свои ресурсы (отсечение по `user_id`).
- Сервисные токены (system) могут понадобиться для импорта/синхронизации данных.
