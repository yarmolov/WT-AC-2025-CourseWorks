# Генерация заданий (.docx) по вариантам

Скрипт: `scripts/generate_assignment_docx.py`

## Что делает
- Берёт список студентов из `students/students.csv` (колонка `Вариант` уже заполнена в файле).
- Берёт описание вариантов из `Курсовые_работы_Веб_Технологии_Варианты_01-40.md`.
- На основе шаблона `BrSTU/Офицальны документы/2_5346014968074496969.docx` создаёт **по одному** `.docx` на студента.

## Установка зависимости
Нужен пакет `python-docx`.

## Запуск
- Режим проверки (ничего не записывает): `--dry-run`
- Обычный режим: генерирует файлы в `WT-AC-2025-CourseWorks/tmp/assignments_docx`

## Как заполняется шаблон
В шаблоне нет явных плейсхолдеров вида `{NAME}`, поэтому скрипт ищет ячейки таблиц по "меткам" и заполняет **соседнюю ячейку справа**:
- `"Тема проекта"` → `"Вариант XX — ..."`
- `"Постановка задачи"` → весь блок варианта в markdown
- `"Дата выдачи"` → дата (по умолчанию сегодняшняя)
- (если найдётся) `"ФИО"` → ФИО студента
- (если найдётся) `"груп"` → группа

Если в вашем файле структура отличается, напишите — подстрою эвристику под ваш шаблон.
